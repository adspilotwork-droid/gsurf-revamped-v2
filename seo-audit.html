Architecture

Single-page site → 16 independent pages

The live site was a single HTML document with anchor-link navigation. Every finding downstream flows from this root cause.

SEO Audit Finding

Severity: High Channels: SEOAEOGEO

The live gsurf.in site was a single HTML page with anchor-link navigation (#home, #about, #plans). Google, Bing and AI crawlers saw one URL with mixed-intent content.

There was no way to rank separately for "enterprise broadband Whitefield", "home broadband Bangalore", or "fibre near ITPL" — only one URL existed.

<!-- All navigation uses #anchor links - one URL for the whole site -->
<a href="#home">Home</a>
<a href="#about">About</a>
<a href="#services">Services</a>
<a href="#plans">Plans</a>
<a href="#contact">Contact</a>
Revamp Implementation

Rebuilt as 16 independent pages — each with its own title, meta description, canonical URL, and structured data. Google can now rank 16 different pages against 16 different queries.

Structured Data

Zero JSON-LD → 53 schemas deployed

The previous site had no machine-readable identity. LocalBusiness schema is now the #1 signal for Google AI Overview inclusion on "near me" queries.

SEO Audit Finding

Severity: High Channels: AEOGEO

The previous HTML contained no JSON-LD, no microdata, no RDFa. Search engines and AI crawlers had no machine-readable business identity, no service definitions, no coverage area.

In 2026, LocalBusiness schema is the single strongest signal for inclusion in Google AI Overviews on "near me" queries.

<head>
  ...
  <!-- no structured data -->
</head>

SEO Audit Finding

Severity: High Channels: SEO

The previous title tag had no city, no service type, no geographic targeting. Brand casing was also inconsistent — "Gsurf" in the title, "GSurf" in body copy.

<title>Gsurf - High-Speed Internet Service Provider | Enterprise Broadband</title>

Every page has a unique, keyword-rich title including service, city, and page intent. Brand rendered consistently as "GSurf" everywhere.

<title>Fibre Broadband in Whitefield (560066) — GSurf Enterprise Fiber</title>
<link rel="canonical" href="https://www.gsurf.in/whitefield.html">
4
Metadata

No meta description → hand-written on all 16 pages

Meta descriptions drive Google snippet quality AND feed AI chatbots as the primary entity summary.

SEO Audit Finding

Severity: High Channels: SEOAEO

No <meta name="description"> tag anywhere. Google auto-generated the search snippet from the first body text it found — "Ultra Fast Internet ! Better Service !!" — with no keywords and no conversion hook.

AI chatbots also pull meta description as the primary entity summary, so the absence of this single tag affects SEO and AEO simultaneously.

Revamp Implementation

Hand-written 150–160 character meta descriptions on all 16 pages, each with primary keyword, city, and value proposition.

<meta name="description" content="Enterprise fibre broadband in Whitefield pincode 560066. Densest coverage, 4-hour fault response, 5-day install, 200+ active customers.">

Semantic HTML

Multiple H1s → exactly one H1 per page

Heading hierarchy tells Google what the page is about. Multiple H1s dilute topical relevance.

SEO Audit Finding

Severity: High Channels: SEO

The previous HTML had multiple <h1> tags on the same page. Google expects one H1 per page as the primary topic declaration. Multiple H1s dilute topical relevance.

<!-- Multiple H1s on the same page (Google expects one) -->
<h1>Ultra Fast Internet ! Better Service !!</h1>
<h1>We Provide Unlimited Network For You.</h1>
<h1>Enterprise Broadband Plans</h1>
Revamp Implementation

Every page in the revamp has exactly one H1. Verified programmatically across all 16 pages.

<h1>Dedicated fiber broadband built for
  <span class="accent">Whitefield's most demanding
  businesses</span>.
</h1>

URL Consolidation

No canonical URLs → absolute canonicals sitewide

Without canonicals, inbound link authority fragments across URL variants (www vs non-www, http vs https).

SEO Audit Finding

Severity: Medium Channels: SEO

No <link rel="canonical"> tags. Without canonicals, Google didn't know whether gsurf.in, www.gsurf.in, http://, or https:// was the preferred URL. Inbound link authority got diluted across variants.

Revamp Implementation

Every page has an explicit canonical pointing to its absolute URL on https://www.gsurf.in. Combined with a host-level 301 redirect rule, all variants consolidate to one canonical set.

<link rel="canonical" href="https://www.gsurf.in/whitefield.html">

No sitemap / robots.txt → both added with AI crawlers opted in

  GSurf
</a>
